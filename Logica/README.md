# Lógica

## Numeros Reais:
```bash
São valores que representam quantidades reais, ou seja, valores que apresentam parte fracionaria, como salario de um funcionario, valor de um boleto, a média de valores, altura e peso de pacientes, etc. Portanto vamos assumir que esses valores podem assumir o comando ponto* decimal. (* como o uso da virgula, agora nas literaturas dizemos separador decimal)

Exemplos: 4.5 1220,12 +15.2 -12,00 + 00,00

```
## Numeros Literal:
```bash
 São valores que representam informações onde se faz necessário utilização de zero ou mais caracteres, tais como nome de uma pessoa, o nome de uma cidade um determinado comprimento, uma frase, etc.
 Eles devem ser especificados entre apóstrofos.


 exemplo: 'Antônio da silva', '1234' , 'rua felipe dutra'

 Outros exemplos:

 limao             > literal
 34434             > numerica
 "verdadeiro"      > literal
 falso             > lógico
"30434"            > literal
-2343              > numérico
"23/12/22"         > literal
verdadeiro         > lógica
"nome"             > literal
```

## Lógico

```bash
São valores que representam as ideias de verdadeiro e falso, onde uma expressão logica oou afirmações são definidas no algoritimo. Por exemplo podemos admitir um valor constante sendocomparado a uma variavel 'a' e feito seu teste: 6<a

Também frases podem ser testadas : A terra gira em torno do sol!

```
